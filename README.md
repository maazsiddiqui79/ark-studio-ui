# 🎨 ARK Studio Landing Page

A clean, modern, and responsive landing page for **ARK Studio**, built using pure HTML and CSS. This project focuses on minimal design, structured layout, and professional UI presentation.

---

## 🚀 Features

- 📌 Hero section with navigation and call-to-action
- 📌 Service/items showcase section
- 📌 Image grid gallery
- 📌 Testimonials section
- 📌 Client branding/logos strip
- 📌 Fully structured footer with useful links
- 📌 Clean and minimal UI design
- 📌 Responsive layout (adaptable to different screen sizes)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3
